📖 什么是禅定

禅那（Dhyana，音译"禅那"）意为"心意的集中"或"冥想"。在传统瑜伽中，它是通往三摩地的七个阶段之一：

持戒 → 体位 → 调息 → 制感 → 专注 → 禅那 → 三摩地

Yama → Asana → Pranayama → Pratyahara → Dharana → Dhyana → Samadhi

🔄 禅定与参究

马哈希虽然以"参究'我是谁'"闻名，但他并不反对禅定：

"禅定是好的，它能平静心智。但禅定本身不是终点。在禅定中，心智变得寂静，但还没有消融。最终，心智必须消融，留下纯粹的觉知——那就是真我。"

— 室利·拉玛那·马哈希

禅定可以看作是参究的准备：

  • 禅定 → 心意专一 → 止息念头
  • 参究 → 追问"谁" → 消融自我感

🧘 禅定的修习

常见的禅定方法：

  1. 专注呼吸 - 把注意力放在呼吸的进出上
  2. 专注咒语 - 持诵神圣音节，如"OM"
  3. 专注心轮 - 把注意力放在心口位置
  4. 专注静默 - 不专注任何对象，只是安住于寂静

"最好的禅定是专注在'我是谁'这个问题上。其他形式的禅定只是热身，最终都要走向参究。"

— 室利·拉玛那·马哈希

✨ 禅定的益处

  • 平静心意，减少焦虑和压力
  • 增强专注力和觉察力
  • 为更深层的修行打下基础
  • 体验到内在的宁静与喜乐
