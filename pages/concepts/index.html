💎 最重要的概念

🔮 真我 / Atman

永恒的、不变的真我本体，是一切众生内在的神性。通过参究"我是谁"可以认识它。

🧠 心智 / Mind

意识的工具，产生念头和感知。了悟真我需要使心智平息。

✨ 恩典 / Grace

上师的加持与神的力量，是觉醒的关键因素。通过参究和虔诚，恩典自然降临。

🌀 梵 / Brahman

宇宙终极实相，一切万有的源头。存在-意识-喜悦（Sat-Chit-Ananda）是梵的本质。

🤫 静默 / Silence

马哈希最标志性的教法——超越言语的深层存在状态，是最高级的教导形式。

👆 上师 / Guru

灵性导师，引导弟子走向自我了悟。马哈希本身就是活生生的上师典范。

🛤️ 参究法 / Jnana

马哈希教导的核心方法——参究"我是谁"，消除无明，认识真我。

🔍 "我是谁？"

马哈希最核心的教法——通过追问这个问题，将心智引向其源头。

🙏 臣服 / Surrender

放下自我意志，将一切归投于神或上师。臣服与参究殊途同归。

👁️ 觉知 / Awareness

纯粹观照的意识，是一切体验的背景。认出觉知就是认出真我。

🕊️ 解脱 / Moksha

从轮回和无明中解脱，证悟真我，是生命的最终目标。

💖 本心 / Heart

真我所居之处，马哈希指出本心在身体右侧胸腔之内，是一切修行的归宿。
