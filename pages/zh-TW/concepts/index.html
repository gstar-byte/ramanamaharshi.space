💎 最重要的概念

🔮 真我 / Atman

永恆的、不變的真我本體，是一切眾生內在的神性。透過參究"我是誰"可以認識它。

🧠 心智 / Mind

意識的工具，產生念頭和感知。了悟真我需要使心智平息。

✨ 恩典 / Grace

上師的加持與神的力量，是覺醒的關鍵因素。透過參究和虔誠，恩典自然降臨。

🌀 梵 / Brahman

宇宙終極實相，一切萬有的源頭。存在-意識-喜悅（Sat-Chit-Ananda）是梵的本質。

🤫 靜默 / Silence

馬哈希最標誌性的教法——超越言語的深層存在狀態，是最高階的教導形式。

👆 上師 / Guru

靈性導師，引導弟子走向自我了悟。馬哈希本身就是活生生的上師典範。

🛤️ 參究法 / Jnana

馬哈希教導的核心方法——參究"我是誰"，消除無明，認識真我。

🔍 "我是誰？"

馬哈希最核心的教法——透過追問這個問題，將心智引向其源頭。

🙏 臣服 / Surrender

放下自我意志，將一切歸投於神或上師。臣服與參究殊途同歸。

👁️ 覺知 / Awareness

純粹觀照的意識，是一切體驗的背景。認出覺知就是認出真我。

🕊️ 解脫 / Moksha

從輪迴和無明中解脫，證悟真我，是生命的最終目標。

💖 本心 / Heart

真我所居之處，馬哈希指出本心在身體右側胸腔之內，是一切修行的歸宿。
