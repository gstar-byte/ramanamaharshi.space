📖 什麼是禪定

禪那（Dhyana，音譯"禪那"）意為"心意的集中"或"冥想"。在傳統瑜伽中，它是通往三摩地的七個階段之一：

持戒 → 體位 → 調息 → 制感 → 專注 → 禪那 → 三摩地

Yama → Asana → Pranayama → Pratyahara → Dharana → Dhyana → Samadhi

🔄 禪定與參究

馬哈希雖然以"參究'我是誰'"聞名，但他並不反對禪定：

"禪定是好的，它能平靜心智。但禪定本身不是終點。在禪定中，心智變得寂靜，但還沒有消融。最終，心智必須消融，留下純粹的覺知——那就是真我。"

— 室利·拉瑪那·馬哈希

禪定可以看作是參究的準備：

  • 禪定 → 心意專一 → 止息念頭
  • 參究 → 追問"誰" → 消融自我感

🧘 禪定的修習

常見的禪定方法：

  1. 專注呼吸 - 把注意力放在呼吸的進出上
  2. 專注咒語 - 持誦神聖音節，如"OM"
  3. 專注心輪 - 把注意力放在心口位置
  4. 專注靜默 - 不專注任何物件，只是安住於寂靜

"最好的禪定是專注在'我是誰'這個問題上。其他形式的禪定只是熱身，最終都要走向參究。"

— 室利·拉瑪那·馬哈希

✨ 禪定的益處

  • 平靜心意，減少焦慮和壓力
  • 增強專注力和覺察力
  • 為更深層的修行打下基礎
  • 體驗到內在的寧靜與喜樂
